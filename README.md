# 连连看游戏

一个基于HTML5/JavaScript的连连看消除游戏，具有卡通风格的精美界面。

## 🎮 游戏特色

- **卡通风格界面**：可爱的渐变色彩和圆角设计
- **流畅连线动画**：可视化连接路径动画效果
- **智能路径算法**：支持直线、一转弯、两转弯连接
- **响应式设计**：适配不同屏幕尺寸
- **emoji图案**：8种精美的emoji方块图案

## 🚀 快速开始

直接在浏览器中打开 `web/index.html` 即可开始游戏！

## 🎯 游戏规则

1. 点击两个相同的emoji方块
2. 连接路径不能超过两个转弯
3. 在规定时间内消除所有方块

## 🎨 技术特色

- **纯HTML5/CSS3/JavaScript**：无需任何依赖
- **现代CSS特效**：渐变、阴影、动画效果
- **面向对象设计**：清晰的代码结构
- **调试友好**：完整的控制台日志

## 📱 兼容性

支持所有现代浏览器：Chrome、Firefox、Safari、Edge

## 🔧 开发

